User Details

Practice / kata portfolio for a volunteer computer & tech Code and Programming Coaches Registry to freely assist career changers and teens seeking to learn programming and web development. Built using an Angular and Bulma CSS front-end, supported by a Node.js, and Express backend built on a JSON Placeholde REStful API. Some assets consumed from RandomUser.me API.
